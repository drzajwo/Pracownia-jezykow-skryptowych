# Space Invaders Lua

Prosta implementacja gry Space Invaders napisana w języku Lua z wykorzystaniem bilbioteki LOVE 2D

### Uruchamianie:
`path/to/love/exe path/to/this/directory`
example:
`/c/Program\ Files/LOVE/love.exe /d/programs/space-invaders`
